Experiment descriptions

Experiment 1

  • Laser: Used
  • Main lamp: Used with a 12h day/night cycle
  • Red lamp: Used for the tank picture
  • Nikon Camera: Used for the PIV
  • Point Grey Camera: Used to take a picture of the tank
  • Traverse: Not used

Created to control the main experiment, there are two timers: 12h and 1h. Every 12h, we switch between day and night. Every hour, we take first a picture of the tank with the Point Grtey, then a visualisation of the flow, ie 10 pictures of the laser sheet with the Nikon with 1s pause. The is no movement of the tank.

Experiment 2

Experiment 4

  • Laser: Used
  • Main lamp: Not used
  • Red lamp: Used for the tank picture
  • Nikon Camera: Used for the flow visualization
  • Point Grey Camera: Used for the tank picture
  • Traverse: Used

Created to control the test experiment in the small tank and in the dark, there is one 1min timer.We take one picture of the tank every minute and a flow visualization every 10min, to do so we move the tank with the traverse twice to scan the whole tank and then go back to the Point Grey height. The tank picutres are taken using the Point Grey camera and the red lamp. The flow visualisation is taken using the laser and Nikon camera
